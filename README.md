# Genban

## 项目概述

Genban 是一个以「应用可视化、模块化」为核心架构的 AI 个人助理系统。系统内置的每个应用都具备双重形态—— AI 可调用的接口，以及用户可视化的界面，双方由此基于同一套应用进行交互，实现人与 AI 的认知对齐。
本项目是 Genban 的服务端项目。

## 核心功能

- **多用户系统**：具备完善的用户管理系统，支持多用户使用
- **多模型支持**：统一对接多家模型平台，支持模型切换
- **应用模块化**：实现模块化的应用管理，支持便捷的应用扩展能力
- **记忆系统**：实现记忆管理机制，具备长期记忆能力

## 初始化（首次运行前）

### 安装环境

- python >= 3.12
- nodejs
- npm
- uv

### 安装 UV

```bash
pip install uv
```

### 创建虚拟环境并安装依赖

```bash
uv sync
```

### 执行初始化脚本

```bash
uv run scripts/init_project.py
```

### 运行服务

```bash
uv run main.py
```
